# Bandgap Reference Circuit Theory

## Introduction

A bandgap reference (BGR) circuit generates a stable voltage reference that is independent of temperature, supply voltage, and process variations. The output voltage is approximately equal to the bandgap voltage of silicon (~1.2V), though scaled versions can produce lower voltages.

## Basic Principle

The BGR combines two voltages with opposite temperature coefficients:

1. **CTAT (Complementary to Absolute Temperature)**: Voltage that decreases with temperature
2. **PTAT (Proportional to Absolute Temperature)**: Voltage that increases with temperature

### Reference Voltage Equation

```
Vref = Vctat + α × Vptat
```

Where α is chosen such that:
```
dVref/dT ≈ 0
```

## CTAT Voltage Generation

### Mechanism
The CTAT voltage is derived from the gate-source voltage (Vgs) of an NFET transistor operating in saturation with constant current.

### Temperature Dependence
```
Vgs = Vth + Vov
```

Where:
- Vth = Threshold voltage (decreases with temperature)
- Vov = Overdrive voltage

The threshold voltage has a negative temperature coefficient:
```
dVth/dT ≈ -0.5 to -2 mV/K
```

### In This Design
- Generated by transistor nfet12
- Biased with constant current
- Temperature coefficient: -231.1275 µV/K

## PTAT Voltage Generation

### Mechanism
The PTAT voltage is generated using the difference in gate-source voltages of two identical transistors operating at different current densities.

### Thermal Voltage
The key parameter is the thermal voltage:
```
VT = kT/q
```

Where:
- k = Boltzmann constant (1.38 × 10⁻²³ J/K)
- T = Absolute temperature (K)
- q = Electron charge (1.6 × 10⁻¹⁹ C)

At room temperature (300K):
```
VT ≈ 26 mV
```

### Current Density Ratio
For two transistors with current ratio N:
```
Vptat = VT × ln(N)
```

### In This Design
- Current ratio N = 8
- Vptat = VT × ln(8) ≈ VT × 2.079
- Generated across resistor R1
- Temperature coefficient: +453.158 µV/K

## Resistor Calculation

### PTAT Resistor (R1)

The PTAT current flows through R1:
```
I = VT × ln(N) / R1
```

For desired current I = 20 µA and N = 8:
```
R1 = VT × ln(8) / I
R1 = 0.026V × 2.079 / 20µA
R1 ≈ 2.7 kΩ (at 300K)
```

Actual design value: R1 = 4.59 kΩ

### CTAT Resistor (R2)

The alpha coefficient relates to the resistor ratio:
```
α = (R2/R1) × ln(N)
```

For α = 0.51 and R1 = 18.711 kΩ:
```
R2 = α × R1 / ln(N)
R2 = 0.51 × 18.711kΩ / 2.079
R2 ≈ 4.59 kΩ
```

## Alpha Coefficient Calculation

For zero temperature coefficient:
```
dVref/dT = dVctat/dT + α × dVptat/dT = 0
```

Solving for α:
```
α = -dVctat/dT / dVptat/dT
α = -(-231.1275 µV/K) / (453.158 µV/K)
α = 0.5100373
```

## Current Mirror Design

### Purpose
- Provides stable bias current
- Replicates current across branches
- Self-biased for startup

### Configuration
- PFET current mirror (top section)
- Multiple branches for current distribution
- Diode-connected reference transistor

### Current Equation
For matched transistors in saturation:
```
I = (W/L) × µCox × (Vgs - Vth)² / 2
```

## Startup Circuit

### Problem
The BGR can have two stable states:
1. Desired operating point (with current flow)
2. Zero-current state (all transistors off)

### Solution
The startup circuit (8 NFETs at bottom) ensures:
- Initial current injection
- Pulls down on critical node
- Disables itself after startup
- Minimal impact on steady-state operation

## Temperature Compensation

### First-Order Compensation
```
Vref = Vctat + α × Vptat
```

This provides linear temperature compensation.

### Higher-Order Effects
Real circuits exhibit curvature due to:
- Non-linear Vth(T) relationship
- Mobility temperature dependence
- Resistor temperature coefficients

### Curvature Compensation (Advanced)
Can be added using:
- Additional PTAT/CTAT terms
- Non-linear resistor networks
- Trimming circuits

## Performance Metrics

### Temperature Coefficient (TC)
```
TC = (1/Vref) × (dVref/dT)
```

Expressed in ppm/K (parts per million per Kelvin).

For this design:
```
TC = (0.14583 mV/K) / (310 mV) × 10⁶
TC ≈ 470 ppm/K
```

### Power Supply Rejection Ratio (PSRR)
```
PSRR = 20 × log₁₀(ΔVdd / ΔVref)
```

Measures sensitivity to supply voltage variations.

### Line Regulation
```
Line Reg = ΔVref / ΔVdd
```

Typically expressed in mV/V or %/V.

## Design Trade-offs

### Voltage vs. Current
- Lower Vref → Lower power
- Higher current → Better noise performance
- Trade-off: Power vs. accuracy

### Area vs. Matching
- Larger transistors → Better matching
- More area → Higher cost
- Trade-off: Area vs. performance

### Speed vs. Stability
- Faster startup → More complex circuit
- Better stability → Larger capacitors
- Trade-off: Speed vs. area

## FinFET Considerations

### Advantages
- Better electrostatic control
- Reduced short-channel effects
- Lower leakage current

### Challenges
- Quantized width (fin count)
- Limited threshold voltage options
- Self-heating effects

### Design Adaptations
- Use nfin parameter for sizing
- Careful thermal management
- Process-specific model parameters

## Conclusion

The bandgap reference achieves temperature independence by:
1. Generating PTAT and CTAT voltages
2. Combining them with proper weighting (α)
3. Using current mirror for stable biasing
4. Including startup circuit for reliability

The result is a stable voltage reference suitable for analog and mixed-signal applications in advanced technology nodes.
