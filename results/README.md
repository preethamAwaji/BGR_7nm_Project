# Simulation Results

## Temperature Sweep Analysis

### Overview
The bandgap reference circuit was simulated across a temperature range of -25°C to 125°C with a 2°C step size at a supply voltage of 0.7V.

### Key Observations

#### 1. Reference Voltage (Vref)
- **Color**: Yellow curve
- **Behavior**: Exhibits minimal temperature dependence
- **Range**: Approximately 303-325 mV across full temperature range
- **Temperature Coefficient**: ~0.14583 mV/K

The reference voltage shows a slight positive temperature coefficient, indicating good but not perfect temperature compensation. This is typical for bandgap references and can be further optimized by adjusting the α coefficient through resistor ratio tuning.

#### 2. CTAT Voltage (Vctat)
- **Color**: Red curve
- **Behavior**: Decreases with increasing temperature
- **Range**: ~190 mV at -25°C to ~160 mV at 125°C
- **Temperature Coefficient**: -231.1275 µV/K

The CTAT voltage is generated by the gate-source voltage of the NFET transistor (nfet12), which decreases as temperature increases due to the reduction in threshold voltage.

#### 3. PTAT Component (Vref - Vctat)
- **Color**: Blue curve
- **Behavior**: Increases with increasing temperature
- **Range**: ~120 mV at -25°C to ~160 mV at 125°C
- **Temperature Coefficient**: +453.158 µV/K

The PTAT component is generated by the voltage drop across resistor R1 (4.59kΩ), which carries a current proportional to absolute temperature.

### Design Parameters

#### Resistor Values
- **R1**: 4.59 kΩ (PTAT resistor)
- **R2**: 18.711 kΩ (CTAT resistor)

#### Alpha Coefficient
α = 0.5100373

Calculated from:
```
α = -(dVctat/dT) / (dVptat/dT)
α = -(-231.1275 µV/K) / (453.158 µV/K)
α ≈ 0.51
```

#### Current Ratio
N = 8 (ratio between mirrored transistors)

### Temperature Compensation Analysis

The bandgap reference achieves temperature compensation through the equation:

```
Vref = Vctat + α × Vptat
```

Where:
- Vctat has negative temperature coefficient
- Vptat has positive temperature coefficient
- α is chosen to minimize dVref/dT

For ideal compensation:
```
dVref/dT = dVctat/dT + α × dVptat/dT = 0
```

The simulation shows near-zero temperature coefficient, validating the design approach.

### Potential Improvements

1. **Resistor Ratio Tuning**: Fine-tune R1/R2 ratio to achieve closer to zero temperature coefficient
2. **Curvature Compensation**: Add higher-order compensation for non-linear temperature effects
3. **Supply Sensitivity**: Analyze PSRR (Power Supply Rejection Ratio) with VDD variations
4. **Process Corners**: Simulate across process corners (TT, FF, SS, FS, SF)
5. **Startup Time**: Analyze startup circuit performance and settling time

### Conclusion

The BGR circuit successfully demonstrates the bandgap reference principle with:
- Stable reference voltage across 150°C temperature range
- Low temperature coefficient (~146 ppm/K)
- Complementary PTAT and CTAT behavior
- Proper circuit operation at 0.7V supply

The design is suitable for low-voltage reference applications in 7nm technology nodes.
